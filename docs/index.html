FindX

Windows 本地文件名搜索引擎：NTFS USN 增量索引、内置中文拼音（全拼 / 首字母 / 子串）、 Everything 风格语法，并兼容 Everything SDK v2 IPC，可被 uTools 等当作 Everything 使用。

下载最新版 源码与文档 反馈问题

亮点

快

Rust 紧凑索引 + USN / MFT，百万级条目仍可保持较低延迟与内存占用。

中文拼音

拼音与排序索引、评分管线打通，适配「只记得读音」的检索习惯。

路径收窄

path: / parent: 与侧栏路径可解析目录子树，候选在子树内再匹配，减少漏搜。

持久化

FXBIN06 含拼音池与 trigram 快照，冷启动以磁盘读为主，避免全量重算。

Everything 生态

WM_COPYDATA / WM_USER 协议与窗口类名兼容，降低迁移成本。

开源

MIT 许可，代码可审、可改、可集成。

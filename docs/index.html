FindX 2.0

本地文件，毫秒级可搜

基于 MFT 与 USN 的 Windows 本地索引与搜索；Everything IPC 兼容；提供独立 CLI、系统服务与 Tauri 图形界面。仓库主线以 Rust 实现，在 GitHub 统一发版与文档。

下载安装包 (Releases) 查看源码 技术说明（README）

核心能力

大规模索引

利用 NTFS MFT 全量建库与 USN 增量更新；可扩展至千万级路径，常规定时搜索可达毫秒级。

与 Everything 协同

服务侧注册 EVERYTHING 兼容窗口与命名管道，便于与现有工具链对接。

多形态入口

findx2 CLI、 findx2-service 常驻、以及桌面 GUI（托盘、热键、设置与预览）。

灵活查询

支持多组过滤、时间/大小、父目录、拼音等扩展与丰富的路径语义。

与 FindX 1.x（.NET）的关系

本页描述的是 2.0 系列（当前 main）。原 .NET 实现已归档在 GitHub 分支 findx-v1，供对照与回退，不再与主线并行发版。
